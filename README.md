# ルールを選べる大富豪

Python 3.12 / Pygameで動く、プレイヤー1人対CPU3人の大富豪です。
タイトル画面からルール設定へ進み、採用するローカルルールを自由に選べます。

## 実装ルール

- ジョーカー（1枚。単体・同じ数字の組札のワイルドカード）
- 8切り
- 革命／革命返し
- スペード3返し
- 縛り
- 階段（同一スート・3枚以上。2とジョーカーは階段に使用しない）
- 5飛び
- 7渡し
- 10捨て
- 11バック
- 禁止上がり
- カード交換（2戦目以降。操作を簡潔にするため自動交換）
- 都落ち（2戦目以降）

## ルール上の定義

大富豪は地域差が大きいため、本作では次のように統一しています。

- 革命は同じ数字4枚、または4枚以上の階段で発生します。
- 11バックはその場が流れるまで有効です。革命中は強弱がもう一度反転します。
- 縛りは、同じスート構成が2回連続したときに発生します。
- ジョーカーは階段の穴埋めには使用しません。
- 7渡し・10捨てでは、人間はカードを選択し、CPUは弱いカードを自動選択します。
- 禁止上がりは、通常時の2、革命・11バック時の3、8、ジョーカーを対象にします。
- カード交換では、大貧民・貧民が強いカードを、大富豪・富豪が弱いカードを自動で渡します。

## 起動方法

```powershell
.\.venv\Scripts\Activate.ps1
python main.py
```

## ファイル構成

- `main.py`：画面表示と操作
- `game_engine.py`：ゲーム進行、CPU、ルール判定
- `rules.py`：ルール設定
- `card.py`：カード定義
- `deck.py`：山札と配札
