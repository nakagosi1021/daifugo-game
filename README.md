# 大富豪（設定・CPU人数・カードデザイン対応版）

Python 3.12 / Pygameで動く大富豪ゲームです。プレイヤー1人に対してCPUを1〜3人まで設定でき、CPU難易度・デモ配札・ローカルルールをゲーム開始前に選べます。

## 今回追加した機能

- CPU人数：1人・2人・3人
- CPU難易度：かんたん・ふつう・むずかしい
- 初期画面をシンプルなメニュー形式へ変更
- J・Q・K・ジョーカーのオリジナル図形イラスト
- CPU手札のカード裏面表示
- カードが手元から場へ移動するアニメーション
- 特殊ルールを確認しやすいデモモード
- 設定内容を `settings.json` に自動保存
- 2〜4人に応じた順位・階級表示
- 人数に応じたカード交換

## 実装済みルール

- ジョーカー
- 8切り
- 革命／革命返し
- スペード3返し
- 縛り
- 階段
- 5飛び
- 7渡し
- 10捨て
- 11バック
- 禁止上がり
- カード交換
- 都落ち

## CPU難易度

- **かんたん**：出せる手からランダムに選択
- **ふつう**：弱いカードを中心に、複数枚や特殊効果も使用
- **むずかしい**：相手の残り枚数、8切り、革命、ジョーカー温存なども考慮

## デモモード

デモモードでは、次の要素を確認しやすい固定配札になります。

- ♦3からの開始
- 階段
- 8切り
- 革命
- ジョーカー
- スペード3返し
- 7渡し、10捨て、11バック

カード交換はデモ配札を崩さないため、デモモード中は実行されません。

## 起動方法

```powershell

python main.py
```

## 簡易テスト

```powershell
python self_test.py
```

## ファイル構成

- `main.py`：画面、設定、操作、カード移動アニメーション
- `game_engine.py`：ゲーム進行、CPU、ルール判定
- `card_art.py`：カード表面・裏面・絵札の描画
- `app_settings.py`：設定の保存と読み込み
- `rules.py`：ローカルルール設定
- `card.py`：カード定義
- `deck.py`：山札と配札
- `self_test.py`：主要機能の簡易テスト

## 標準プリセットのカスタマイズ

初期状態の「標準」は、13個のローカルルールのうち7個がONです。

1. ゲーム設定から「ローカルルール詳細」を開く
2. 好きなルールをON／OFFにする
3. 「現在の組み合わせを標準に登録」を押す
4. 設定画面へ戻り、「保存してタイトルへ」を押す

次回から「標準（編集可）」または「登録済み標準」を押すと、保存した組み合わせが適用されます。

「標準を初期状態（7個）に戻す」を押すと、最初の標準設定へ戻せます。

## 手札の明暗表示

自分の手番では、現在の場に合法的に出せる手に含まれるカードだけが
明るく表示されます。カードを1枚選ぶと、その選択と組み合わせて
合法手を完成できるカードだけが引き続き明るく表示されます。

暗いカードは現在の選択では出せないため、クリックできません。

## 禁止上がり

本作では2上がり、および革命中の3上がりは許可しています。
禁止上がりの対象は、8切りが有効な場合の8上がりとジョーカー上がりです。
